# China Trip Knowledge Base

This repository is a Markdown/Obsidian-friendly knowledge base for the China trip planned for 2026-06-19 through 2026-07-01.

It consolidates route planning, daily schedules, bookings, tickets, transport, risks, packing, costs, accepted decisions, rejected alternatives, and open checks. It is intended to be usable by both Codex and ChatGPT as the Single Source of Truth for the trip.

## How To Use

Start with [INDEX.md](./INDEX.md), [CURRENT_STATE.md](./CURRENT_STATE.md), [DAILY_PLAN.md](./DAILY_PLAN.md), [DECISIONS.md](./DECISIONS.md), and [RISKS_AND_CHECKS.md](./RISKS_AND_CHECKS.md).

## Main Files

- [ROUTE.md](./ROUTE.md) — overall route and sequence.
- [BOOKINGS.md](./BOOKINGS.md) — hotels and accommodation state.
- [TICKETS.md](./TICKETS.md) — parks, scenic areas, and attraction tickets.
- [TRANSPORT.md](./TRANSPORT.md) — flights, trains, taxis, difficult transfers.
- [PLACES.md](./PLACES.md) — place reference.
- [OPEN_QUESTIONS.md](./OPEN_QUESTIONS.md) — unresolved planning questions.
- [PACKING_AND_GEAR.md](./PACKING_AND_GEAR.md) — luggage, shoes, clothing, mountain gear.
- [COSTS.md](./COSTS.md) — confirmed and open costs.
- [RAW_CONTEXT.md](./RAW_CONTEXT.md) — cleaned full source context.
- [archive/original_input.md](./archive/original_input.md) — sanitized archival copy of the original master input.

Sensitive personal data, exact order numbers, emails, phone numbers, QR codes, and passport fields are masked or omitted where present.
